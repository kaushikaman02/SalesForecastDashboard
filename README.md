# Sales Forecast Dashboard

## 📌 Project Overview
This is an interactive Sales Forecast Dashboard built using Streamlit. It provides sales insights, forecasting, anomaly detection, and product demand segmentation.

## 🚀 Features
- Sales Overview Dashboard
- Monthly Sales Trend
- Region & Category Filters
- Forecast Explorer
- Anomaly Report
- Product Demand Segmentation (K-Means Clustering)

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
